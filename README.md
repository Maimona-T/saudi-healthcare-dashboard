# saudi-healthcare-dashboard
A 2-page Power BI dashboard analyzing hospital capacity, staffing, and patient metrics across all 13 Saudi regions, with infrastructure growth trends from 2015 to 2023.
Overview


Page 1 — Regional Overview 2023: filled map of bed capacity per 1,000 population, doctor density comparison by region, and key national KPIs
Page 2 — Infrastructure Growth 2015–2023: national workforce and bed capacity trends, per-capita ratios vs. OECD benchmarks, and regional disparity between urban (Riyadh, Makkah) and remote regions (Najran, Northern Borders, Al-Baha)

Data Source
